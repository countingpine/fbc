va_first
 
Returns the first argument in a variable argument list

Syntax

pointer_variable = va_first()

Description

The va_first function initializes a pointer, pointer_variable, to point to the first variable argument passed to a function.

Example

Function Avg cdecl (Count As Integer, ... ) As Double
    Dim ARG As Any Ptr 
    Dim SUM As Double = 0
    Dim i As Integer
    
    ARG = va_first()

    For i = 1 To COUNT
        SUM += va_arg(ARG, Double)
        ARG = va_next(ARG,Double)
    Next i
    Return SUM/COUNT
End Function

Print AVG (4, 3.4,5.0,3.2,4.1)
Print AVG (2, 65.2,454.65481)

Sleep


The output would look like:
3.925
259.927405
