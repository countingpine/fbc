Compiler Option: -r
 
Compile into asm file(s) only, do not assemble or link

Syntax

-r

Description

The -r option specifies that any source files listed are to be compiled to asm files, and not compiled or linked into an executable. Note that in case of -gen gcc *.c files are generated, instead of the normal *.asm files.
