__FB_VER_MAJOR__
 
Intrinsic define (macro value) set by the compiler

Syntax

__FB_VER_MAJOR__

Description

__FB_VER_MAJOR__ will return the major version of FreeBASIC currently being used. For example, the major version is 0 for Freebasic 0.18, and will remain 0 until FreeBASIC 1.0 is released.

Example

Dim fbMajorVersion As Integer
Dim fbMinorVersion As Integer

fbMajorVersion = __FB_VER_MAJOR__
fbMinorVersion = __FB_VER_MINOR__

Print "Welcome to freebasic ";fbMajorVersion;".";fbMinorVersion
