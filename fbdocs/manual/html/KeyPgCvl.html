CVL
 
Converts an four byte string to an integer (long) variable

Syntax

Declare Function CVL ( ByRef str As Const String ) As Integer

Usage

result = CVL( str )

Parameters

str
A String exactly four bytes in length with a binary copy of an integer variable stored in it.

Return Value

A Integer variable to copy the binary copy of a integer to.

Description

Does a binary copy from a 4-byte-string to a long variable. The result will have a sense only if the string contained a correctly formatted integer, as the one generated by MKL or MKI.

This function is useful to read numeric values from buffers without using a Type definition.

Example

Dim a As Integer, b As String
a=4534
b=MKI(a)
Print a, CVI(b)
