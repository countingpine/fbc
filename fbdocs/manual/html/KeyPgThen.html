Then
 
Control flow statement for conditional branching.

Syntax

If expression Then statement(s) [Else statement(s)]
or
If expression Then : statement(s) [Else statement(s)] : End If
or
If expression Then
statement(s)
[ ElseIf expression Then ]
statement(s)
[ Else ]
statement(s)
End If
