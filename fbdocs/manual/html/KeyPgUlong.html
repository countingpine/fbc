Ulong
 
Standard data type: unsigned integer having same size as SizeOf(Any Ptr)

Syntax

Dim variable As Ulong

Description

Depending on the platform, same as UInteger or ULongInt. A 32-bit or 64-bit unsigned whole-number data type.

Ulong has the same size as SizeOf(Any Ptr).

Example

  Dim x As ULong = 0
  Dim y As ULong = &HFFFFFFFF
  Print "ULong Range = "; x; " to "; y

Output:
ULong Range = 0 to 4294967295

Dialect Differences

  • Not available in the -lang qb dialect unless referenced with the alias __Ulong.

Differences from QB

  • New to FreeBASIC
