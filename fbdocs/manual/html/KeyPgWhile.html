While
 
Control flow statement.

Syntax

Do While condition
[statement block]
Loop
or
Do
[statement block]
Loop While condition
or
While [condition]
[statement block]
Wend

Description

While specifies that a loop block will continue if the condition following it evaluates as true. This condition is checked during each loop iteration.
